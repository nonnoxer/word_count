Word Count

(for PW and more)


regex (see regexr) to ignore:


Word Count: {{words}}

What is this?

Helps you count only your body paragraphs of a document

Ignores headers, references and in text citations!

How to Use

Copy to a docx file and upload it here (only docx accepted)

Make sure that paragraphs you want to count have a first line indent (not hanging)

Edit the regex to be excluded: default regex \([^()]*\, ?[0-9]*\) matches in-text citations (name, year)

Other regex options: \([^()]*\) to ignore any brackets

Notes

Files are IMMEDIATELY DELETED after counting, your files are not stored
