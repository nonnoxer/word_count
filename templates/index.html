Word Count

(for PW and more)

Word Count: {{words}}

What is this?

Helps you count only your body paragraphs of a document

Ignores headers, references and in text citations!

How to Use

Save/Copy to a docx file and upload it here (only docx accepted)

Make sure that paragraphs you want to count have a first line indent (not hanging)

Make sure that in text citations have the format (name, year)
